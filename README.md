Sales Analysis Report Dashboard (Power BI)

A comprehensive Sales Analysis Dashboard built using Power BI to visualize and analyze sales performance across products, customers, orders, and geographic locations. The dashboard provides interactive insights that help identify top-performing products, customer purchasing trends, sales distribution by state, and overall business performance metrics.
